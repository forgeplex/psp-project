# PSP Project Hub

PSP（Platform Service Portal）项目的总体信息中心。

由 **Alex（Chief）** 和 **Atath（PM）** 共同维护。

## 文件索引

| 文件 | 内容 | 维护人 |
|------|------|--------|
| `PROJECT.md` | 项目概览、技术栈、环境信息、仓库列表 | Alex |
| `PROGRESS.md` | 模块进度看板 | Alex + Atath |
| `ROADMAP.md` | 项目规划与里程碑 | Atath |
| `CHANGELOG.md` | 重要变更记录 | Alex |

## 使用方式

所有 Agent 在新 session 启动时应 `git pull` 此仓库，读取 `PROJECT.md` 获取项目基础信息。
